Scope

  • The glue between the application controller and the view (HTML)
  • It's (basically) just a JavaScript object
  • It's the context Angular expressions are executed in

Modules

  • Allow us to break up our application
  • Give us access to a lot more of Angular's features

Controller

  • Controls flow of data within our application
  • Accepts $scope as a parameter, which refers to the application/module that the controller needs to handle

Let's walk through that

In index.html:

  1. Include a script tag after the angular script tag to include App.js
  2. Set ng-app to the name of your module
  3. Add ng-controller and set to the name of your controller
  4. Access values provided in controller

In App.js:

  1. Create a module
  2. Create a controller in the module
